# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-05

### Added
- Complete **Redcode MARS Engine** supporting 15 opcodes.
- Interactive **Battle Arena** with memory visualization and event logging.
- **Redcode Editor** with syntax highlighting and autocomplete.
- **Tech Tree** progression system in the Academy.
- **Chaos Mode** and customizable battle parameters.
- Multiplatform support for **Android** and **Web (Wasm)**.
- Professional project documentation (README, CONTRIBUTING, etc.).
- Comprehensive unit test suite for the engine and parser.

### Changed
- Migrated all dependencies to early 2026 stable versions.
- Modernized build system with Version Catalogs and new Kotlin DSLs.
- Improved performance of batch simulation.
